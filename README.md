# Capture — Releases

Public download host for **Capture** — a free Windows screen recorder with smart
image capture, GIF/MP4 export, a webcam overlay, and AI (or your‑own) voiceover.

🌐 **Website:** https://capture.deb0.com

## Download

**[⬇ Download the latest installer (Capture‑Setup.exe)](https://github.com/DebajyotiSaikia/capture-release/releases/latest/download/Capture-Setup.exe)**

Windows 10/11 · free. The installer is unsigned, so Windows SmartScreen may warn
on first run — choose **More info → Run anyway**.

## Features

- **Region capture** — drag to select any area (with aspect‑ratio lock) and
  start recording right from the selection overlay.
- **Smart image capture** — saves a frame only when the screen changes
  (adjustable sensitivity, optional mouse tracking).
- **Video & GIF** — record to MP4, or build a video/GIF from images with
  per‑image durations.
- **Webcam overlay** — put your camera in any corner with optional **background
  blur** and **face‑tracking auto‑zoom**, burned into recordings and your‑voice
  videos. Off by default.
- **Voiceover** — AI neural narration time‑fitted to the video, or record your
  own mic; pick your microphone and camera.
- **Multimodal AI** — a vision model reads your frames to write the script and
  timed captions (.srt). OpenAI, Anthropic, GitHub Models & Copilot.
- **Themes & polish** — system / light / dark, a frameless window, and a tested
  codebase.

See everything on the website → **https://capture.deb0.com**

## Privacy

Everything runs locally — recordings, images and settings stay on your device.
AI features only call the provider you configure, using your own key.
